# 🚀 Job Portal System

## 📌 Project Overview

The **Job Portal System** is a full-stack web application designed to connect **job seekers and recruiters** on a single platform. It allows companies to publish job opportunities while enabling candidates to search, apply, and track their job applications.

This system aims to simplify the recruitment process by providing a centralized platform where job seekers can explore opportunities and recruiters can efficiently manage applicants.

---

# 🎯 Project Objectives

* Provide a centralized platform for job postings and applications
* Enable recruiters to manage job listings and applicants
* Allow candidates to search and apply for jobs easily
* Implement a scalable backend architecture
* Build a responsive and user-friendly frontend interface

---

# 👥 User Roles

### 🧑‍💻 Job Seeker

* Register and create a profile
* Search jobs by location or company
* Apply for jobs
* Upload resume
* Track application status

### 🏢 Recruiter

* Post new job openings
* Update or delete job listings
* View candidate applications
* Shortlist or reject applicants

### ⚙️ Admin

* Manage users and job postings
* Monitor platform activity
* Maintain system integrity

---

# 🧩 Core Modules

### 👤 User Management

Handles user registration, login, and profile management.

### 💼 Job Management

Allows recruiters to create and manage job postings.

### 📄 Application Management

Manages job applications submitted by candidates.

### 🔍 Job Search

Provides job search functionality with filters like location, role, and company.

### 📊 Dashboard

Provides recruiters and admins with analytics and insights.

# 🏗 System Architecture

Frontend → Backend → Database

```
React (Frontend)
      |
REST API Communication
      |
Spring Boot (Backend)
      |
Spring Data JPA
      |
MySQL Database
```
